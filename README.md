# sotabkw.me

Build with

- Next.js<br>
- Typescript<br>
- TailwindCSS<br>

# install dependencies

```bash
yarn install
```

## start dev server

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
